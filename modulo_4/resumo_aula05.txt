COMANDOS: 

GIT LOG -  MOSTRA INFORMAÇÕES DE ALTERAÇÕES E SEU AUTOR. 

GIT LOG --DECORATE -  MOSTRA INFORMAÇÕES COMO DE-PARA AONDE.

GIT LOG --AUTHOR="EXEMPLO" -  MOSTRA INFORMAÇÕES DO AUTOR QUE TENHA ESSA INFORMAÇÃO.

GIT SHORTLOG - MOSTRA QUAL FOI O AUTOR, QUANTOS COMMIT FORAM FEITOS E QUAIS FORAM. 

GIT SHORTLOG -SN - MOSTRA SÓ A QUANTIDADE DE COMMIT E O AUTOR. 
 
GIT LOG --GRAPH - MOSTRA EM FORMA GRAFICA O QUE ESTÁ ACONTECENDO COM MINHAS VERSÕES.

MODULO 04 - AULA 05 - 12. Desfazendo coisas com o reset - Git e Github para Iniciantes

GIT CHECKOUT - RETORNA O ARQUIVO PRA ANTES DA EDIÇÃO, ATENÇÃO: ANTES DE TER SIDO ADICIONADO. 

GIT RESET HEAD - DEPOIS DE TER ADICIONADO, PODE SER UTILIZADO PARA RETORNO AO STATUS ANTES DE ADICIONAR E DEPOIS BASTA UTILIZAR O GIT CHECKOUT QUE ESTORNA O QUE FOI FEITO. 

GIT RESET --SOFT - VAI VOLTAR E MATAR O COMMIT QUE FOI REALIZADO.
GIT RESET --MIXED - VAI MATAR O COMMIT E VOLTAR OS ARQUIVOS PRA ANTES DO STAGED 
GIT RESET --HARD - VAI MATAR TUDO QUE FOI FEITO DO COMMIT.
