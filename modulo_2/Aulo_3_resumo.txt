AULA 3 - VÍDEO O que é API? REST e RESTful? aprenda com Star Wars!  


	
	API(APPLICATION PROGRAMMING INTERFACE / INTERFACE DE PROGRAMAÇÃO DE APLICAÇÕES) - "UM CONJUNTO DE ROTINAS E PADRÕES ESTABELECIDOS POR UM SOFTWARE PARA A 
										           UTILIZAÇÃO DAS SUAS FUNCIONALIDADES POR APLICATIVOS QUE NÃO PRETENDEM 
											   ENVOLVER-SE EM DETALHES DA IMPLEMENTAÇÃO DO SOFTWARE"

		OBS: ESTRUTURAS DE DADOS JSON/XML


       REST - SÃO PRINCIPIOS DE ARQUITETURA, PARA OS DESENVOLVEDORES CRIAREM SUAS APIS. 
       RESTFULL - USA OS PRINCIPIOS DE ARQUITETURA RECOMENDADOS PARA CONSTRUÇÃO DE UMA API.
